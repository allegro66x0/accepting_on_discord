# Remote Approver Bot

このツールは、PCの画面を監視して特定のボタン（OK/承認など）が表示されたらDiscordに通知＆遠隔操作するためのボットです。

## 使い方（友達へ）

### 1. 準備
1. このフォルダをダウンロード・解凍します。
2. `AoS` フォルダの中にある `config.json.sample` というファイルの名前を `config.json` に変更します。
3. `config.json` をメモ帳などで開き、以下の項目を設定します。
   - `DISCORD_TOKEN`: あなたのDiscord Botのトークン
   - `CHANNEL_ID`: 通知を送りたいチャンネルのID
   - `TARGET_IMAGES`: 監視したいボタンの画像ファイル名（自分でスクショしてAoSフォルダに入れてください）

> **注意**: `config.json` は「秘密の鍵」なので、絶対に他人に渡したりネットに上げたりしないでください！

### 2. 実行
1. `launch_approver.bat`（または `.lnk` ショートカット）をダブルクリックして起動します。
2. 黒い画面（コマンドプロンプト）が出れば起動成功です。
3. 最初の状態は「待機モード」です。

### 3. 操作コマンド (Discord)
Discordのチャンネルで以下のチャットを送ると操作できます。

- `!start`: 画面監視をスタートします。（「👀 監視を開始しました」と返事があればOK）
- `!stop`: 監視を止めます。

### 4. 通知が来たら
- ボタンが見つかるとスクショ付きで通知が来ます。
- 「承認」ボタンを押すと、PC側で自動的にクリックされます。
